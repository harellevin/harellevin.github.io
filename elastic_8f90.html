Calculates stress tensor for elastic strains.

Test for convexity and calculates stress s=(sxx,syy,szz,sxy,sxz,syz) for stable element Define the flag value according to the rule: flg = 0 - stable flg = 1 - 1-st type of failure flg = 2 - 2-nd type of failure flg = 3 - open space

calculates second invariant i2(n) and strain invariant ratio ksi(n) event = true means failed element

Parameters:
[in] n - element number
[out] event=true/false; s - stress tensor
