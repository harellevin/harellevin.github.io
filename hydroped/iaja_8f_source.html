00127 C Subroutine for assembling element matrices.
00128 C The resulting matrix has a symmetric sparsity pattern but may have
00129 C unsymmetric values. Optionally assembles only the sparsity pattern.
00130 C
00131 C Argument list (* indicates may be changed by rouitne)
00132 
00133 C LCNTL : LOGICAL array of controlling parameters of length 10. The user
00134 C         must set LCNTL(1) to LCNTL(4).
00135 C LCNTL(1) : must be TRUE if symmetric and only lower
00136 C            triangle wanted.
00137 C LCNTL(2) : should be set to TRUE if user would like the entries
00138 C            within each column to be ordered by rows.
00139 C LCNTL(3) : should be set to TRUE if user wants null rows removed.
00140 C LCNTL(4) : should be set to FALSE if only sparsity pattern wanted.
00141 C         The other entries of LCNTL are not currently accessed.
00142 C  NMAX:  INTEGER. On entry, must be at least as large as the
00143 C         largest integer used to index
00144 C         a variable.
00145 C NELT :  INTEGER. Must be set to number of elements.
00146 C ELTVAR :INTEGER array. On entry, leading
00147 C         entries must hold element index lists. Unchanged on exit.
00148 C ELTPTR :INTEGER array length NELT+1
00149 C         On entry, must hold pointers for ELTVAR.
00150 C AELT :  REAL (DP) array.  LCNTL(4) = TRUE, on entry,
00151 C         element entries must be in AELT, with the entries for
00152 C         each element stored column-by-column. If LCNTL(1) = TRUE ,
00153 C         only entries in lower triangular part of element matrix
00154 C         should be stored in packed form.
00155 C         Not accessed if LCNTL(4) = FALSE.
00156 C *N   :  INTEGER. Not set on entry.
00157 C         On exit, holds the order of the assembled matrix
00158 C LIRN  : INTEGER. length of array IRN. Note that, because of
00159 C         duplicates, must be larger than number of entries in final
00160 C         assembled matrix.
00161 C *IRN :  INTEGER array length LIRN. Not set on entry. On exit
00162 C         holds row indices of entries in assembled matrix, ordered
00163 C         by columns. If SYM = .TRUE., only lower triangular part
00164 C         is stored.
00165 C *IP  :  INTEGER array length at least NMAX+1.
00166 C         Not set on entry. On exit, first N+1 entries hold col.
00167 C         pointers for assembled matrix.
00168 C LA  :   INTEGER. Length of array A. LA must be set to 1 if
00169 C         only pattern of A required.
00170 C *A   :  REAL (DP) array length LA.
00171 C         Not set on entry. If LCNTL(4) = TRUE,
00172 C         on exit holds entries of assembled matrix (ordered by rows).
00173 C         Not accessed if LA = 1.
00174 C *IW   : INTEGER array length at least 2*N.
00175 C         On exit, IW(J) is the original
00176 C         variable index for row (or column) J in assembled matrix.
00177 C LP    : INTEGER. Output stream for error messages.
00178 C *INFO : INTEGER array length 10. INFO(1) used as error flag.
00179 C         If INFO(1) = -3 on exit INFO(2) holds minimum value for LIRN
00180 C         If INFO(1) = -4 on exit INFO(2) holds minimum value for LA
00181 C         INFO(3) holds largest integer used to index a variable.
00182 C         INFO(4) number of entries in assembled matrix.
00183 C         INFO(5) holds first element with duplicated indices.
00184 C         INFO(6) holds number of null rows/cols (removed if
00185 C         LCNTL(3) = TRUE).
00186 C         INFO(7) holds smallest integer used to index a variable.
